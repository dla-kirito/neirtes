# Vue.js Modal Component
## Overview
- Example of a modal component in Vue.js

## Key Points
- **CSS Classes**
  - `modal-mask`
  - `modal-wrapper`
  - `modal-container`
- **Customization**
  - Modify CSS styles for header, body, and footer
- **Transition Effects**
  - Controlled by classes `modal-enter` and `modal-leave-to`
